# YAML : yet another markup language

- Key value pair 

    ```
    name: 'my name'
    or 
    name: "your name"
    or 
    name: your name
    ```
- Dictionary 
    ```
    name: 
        label: my name 
        description: name description 
        environment: hot 
    ```

- Array 
    ```
    colors: 
        - red 
        - green 
        - blue
    ```
